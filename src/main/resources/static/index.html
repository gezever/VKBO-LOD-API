DEMO LOD API ONDERNEMINGEN

Ondernemingen en vestigingen API - VKBO OGC van JSON naar JSON-LD en Turtle

Deze webapplicatie stelt een eenvoudige HTTP-API beschikbaar waarmee je informatie over een Onderneming (op basis van ondernemingsnummer en vestigingsnummer) kunt opvragen in verschillende formaten:

  • application/json (originele API-response)
  • application/ld+json (verrijkte JSON-LD)
  • text/turtle (geparsed RDF-model in Turtle)
  • application/rdf+xml (geparsed RDF-model in rdf/xml)
  • text/html

Het gewenste formaat kan opgevraagd worden via content negotiatie:

  • Ofwel via de HTTP ACCEPT header
  • Ofwel via de query parameter 'format' met 1 van de volgende waarden: json, jsonld, ttl, rdf
